This repository tracks and analyzes water quality changes over time across 22 catchment facilities from 2011 to 2024. The project provides insights into long-term trends, helping stakeholders monitor environmental health and make data-driven decisions.

Features

Objective

Empower environmental researchers, policy makers, and water management authorities with actionable insights to improve water quality management and sustainable resource planning.
