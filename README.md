# Maps App

Minimal Vite + React + TypeScript starter.

## Quick start

Install dependencies and run the dev server:

```bash
cd maps-app
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Notes

- `src/map.tsx` is a placeholder component where you can integrate mapping libraries (Leaflet, Mapbox, Google Maps, etc.).
